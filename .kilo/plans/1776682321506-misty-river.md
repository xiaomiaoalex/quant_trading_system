### Root Cause
In `strategies.py`, `_oms_handler` is set to the callback **function wrapper**, not the `OMSCallbackHandler` **instance**. When `get_oms_metrics()` calls `hasattr(_oms_handler, 'get_dedup_stats')`, it returns `False` because the wrapper function doesn't have this method.

### Files to Fix
1. **`trader/api/routes/strategies.py`**:
   - Add `_oms_handler_instance` global to store the actual handler
   - Update `_get_oms_handler()` to set `_oms_handler_instance = handler`
   - Update `get_oms_metrics()` to use `_oms_handler_instance`

```python
# Add global (around line 121)
_oms_handler_instance: Optional[Any] = None

# In _get_oms_handler() after line 146:

# In get_oms_metrics() (lines 157-168):

class OMSCallbackHandler:
    def __init__(self, ...):
        self._order_submit_ok: int = 0
        self._order_submit_reject: int = 0
        self._order_submit_error: int = 0
        self._reject_reasons: Dict[str, int] = {}
        self._fill_latencies: List[float] = []  # ms
    
    def record_submit(self, ok: bool, reason: Optional[str] = None):
        if ok:
            self._order_submit_ok += 1
        elif reason:
            self._order_submit_reject += 1
            self._reject_reasons[reason] = self._reject_reasons.get(reason, 0) + 1
        else:
            self._order_submit_error += 1
    
    def record_fill_latency(self, latency_ms: float):
        self._fill_latencies.append(latency_ms)
        if len(self._fill_latencies) > 1000:
            self._fill_latencies = self._fill_latencies[-500:]  # keep recent
    
    def get_runtime_stats(self) -> dict:
        latencies = sorted(self._fill_latencies)
        p50_idx = int(len(latencies) * 0.5)
        p95_idx = int(len(latencies) * 0.95)
        return {
            "order_submit_total_ok": self._order_submit_ok,
            "order_submit_total_reject": self._order_submit_reject,
            "order_submit_total_error": self._order_submit_error,
            "reject_reason_counts": dict(self._reject_reasons),
            "fill_latency_ms_p50": latencies[p50_idx] if latencies else 0,
            "fill_latency_ms_p95": latencies[p95_idx] if latencies else 0,
        }
```

#### Step 19.3: Add tick metrics tracking
**File**: `trader/services/strategy_runtime_orchestrator.py`

```python
class StrategyRuntimeOrchestrator:
    def __init__(self, ...):
        self._tick_timestamps: List[float] = []
        self._tick_rate: float = 0.0
        self._tick_lag_ms: float = 0.0
    
    def record_tick(self, tick_timestamp: float):
        now = time.time()
        self._tick_timestamps.append(now)
        # Keep last 100 timestamps
        if len(self._tick_timestamps) > 100:
            self._tick_timestamps = self._tick_timestamps[-100:]
        # Calculate rate (ticks/sec over window)
        if len(self._tick_timestamps) >= 2:
            window = now - self._tick_timestamps[0]
            self._tick_rate = len(self._tick_timestamps) / window if window > 0 else 0
        self._tick_lag_ms = (now - tick_timestamp) * 1000
```
